# 🏭 대체연료본부 가스연료기술센터 — 사업 현황 대시보드

엑셀 파일을 업로드하면 **국가연구개발사업 · 수탁용역 · 시험인증** 데이터를 자동으로 분석해 차트와 표로 보여주는 웹 대시보드입니다.

> 🔒 **개인정보/보안**: 업로드한 엑셀 파일은 **브라우저 안에서만** 읽히고 처리됩니다. 서버나 GitHub 저장소 어디에도 **저장되거나 전송되지 않습니다.**

---

## 🚀 사용 방법

1. 대시보드 페이지에 접속합니다. (아래 "배포" 참고)
2. **📁 엑셀 파일 선택** 버튼을 눌러 사업 현황 엑셀 파일을 업로드합니다.
3. 자동으로 KPI · 차트 · 표가 그려집니다.
4. 상단의 **구분 필터**(전체/기존/연장/신규/기획)로 국가연구개발사업 데이터를 걸러볼 수 있습니다.

---

## 📊 대시보드 구성

| 영역 | 내용 |
|------|------|
| **KPI 카드** | 총 과제 수, 팀별(전산·탄소·극저온) 실적 합계 |
| **팀별 실적 합계** | 막대그래프 |
| **팀별 과제 수** | 막대그래프 |
| **센터 실적현황** | 3개 팀 합계 및 비율 (원형차트 + 팀별 카드) |
| **전체 과제 목록** | 시트별 탭 표 (국가연구개발사업 / 수탁용역 / 시험인증) |

---

## 📁 엑셀 파일 요구사항

다음 **3개 시트**가 있어야 합니다. (시트 이름이 정확히 일치해야 합니다.)

- `국가연구개발사업`
- `수탁용역`
- `시험인증`

> `분기별 보고` 시트는 읽지 않습니다. (대시보드 수치와 **비교용**으로 별도 활용)

### 💰 금액(실적) 기준

| 시트 | 실적으로 사용하는 값 |
|------|------------------|
| 국가연구개발사업 | **당해년도 사업비 합계** (인건비 + 간접비) |
| 수탁용역 | **총 인정실적(A+B)** |
| 시험인증 | **시험수수료(A)** — 전부 **극저온 팀**으로 집계 |

### 규칙

- **헤더 자동 인식**: 제목 행이나 2단(병합) 헤더가 있어도 키워드로 헤더 행을 자동 탐지합니다.
- **합계 행 자동 제외**: 시트 맨 아래 "합계/총계/소계" 행은 제외하고 직접 재합산합니다.
- **빈 값 = 0원 처리**: `미정`, `협약전`, `미승금`, `-`, 빈칸 등 **숫자로 변환할 수 없는 값은 모두 0원**으로 계산됩니다. (나중에 값이 채워지면 자동 반영)
- **구분 필터**: `기존/연장/신규/기획` 필터는 **국가연구개발사업 시트에만** 적용됩니다. (수탁용역·시험인증은 항상 포함)

## 🌐 배포 (GitHub Pages)

1. 저장소 **Settings → Pages** 로 이동
2. **Source**: `Deploy from a branch`
3. **Branch**: `main` / `/ (root)` 선택 후 **Save**
4. 잠시 후 `https://doo-mo.github.io/Gas-center-dashboard/` 에서 접속

> ⚠️ 이 저장소는 **Private** 입니다. Private 저장소에서 GitHub Pages를 쓰려면 **유료 플랜(GitHub Pro 등)** 이 필요할 수 있습니다. 무료 플랜이라면 저장소를 Public으로 전환하거나, 로컬에서 `index.html`을 브라우저로 직접 열어 사용할 수 있습니다.
