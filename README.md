# 🏭 대체연료본부 가스연료기술센터 — 사업 현황 대시보드

엑셀 파일을 업로드하면 **국가연구개발사업 · 수탁용역 · 시험인증** 데이터를 자동으로 분석해 **KPI · 차트 · 표**로 보여주는 웹 대시보드입니다.

> 🔒 **개인정보/보안**: 업로드한 엑셀 파일은 **브라우저 안에서만** 읽히고 처리됩니다. 서버나 GitHub 저장소 어디에도 **저장되거나 전송되지 않습니다.**

---

## 🚀 사용 방법

1. 대시보드 페이지에 접속합니다. (아래 "배포" 참고)
2. **📁 엑셀 파일 선택** 버튼을 눌러 사업 현황 엑셀 파일을 업로드합니다.
3. 자동으로 KPI · 차트 · 표가 그려집니다.
4. 필요하면 우측 상단의 **🖼️ 화면 캡처** 또는 **📄 PDF 내보내기** 버튼으로 결과를 저장합니다.

---

## 📊 대시보드 구성

| 영역 | 내용 |
|------|------|
| **KPI 카드 (4개)** | 센터 전체 실적(+목표금액·달성률), 전산팀·탄소팀·극저온팀 실적 (각 카드에 국가/수탁/시험 항목별 내역 표시) |
| **센터 실적현황 — 팀 비율** | 도넛 차트 — 3개 팀 비율 + **가운데에 목표 달성률(%)** 표시 |
| **센터 실적현황 — 사업 항목 비율** | 도넛 차트 — 국가연구개발사업 / 수탁용역 / 시험인증 **3개 항목 비율** + 가운데에 센터 합계 표시 |
| **팀별 실적 합계** | 누적 막대 차트 — 팀별(전산팀/탄소팀/극저온팀)로 국가/수탁/시험 실적을 쌓아서 표시 (막대에 마우스를 올리면 **팀 내 비율(%)**과 팀 합계 표시) |
| **전체 목록** | 시트별 탭 표 (국가연구개발사업 / 수탁용역 / 시험인증), 맨 위에 합계 행 고정 |

> 차트 3개는 가로로 나란히 배치되며, 화면이 좁아지면 자동으로 세로로 쌓입니다.

## 🎯 목표금액 · 달성률

- **목표금액**은 `분기별 보고` 시트에서 **"운용자금 목표금액"** 글자를 찾아 그 아래(또는 옆) 칸의 숫자를 자동으로 읽어옵니다.
- **달성률(%) = 센터 전체 실적 ÷ 목표금액 × 100**
- 목표금액을 찾지 못하거나 `분기별 보고` 시트가 없으면, 목표·달성률 표시는 **자동으로 숨겨집니다.** (팀 비율 도넛 가운데에는 대신 센터 합계 금액 표시)

## 🖼️ 내보내기 (캡처 / PDF)

- **화면 캡처**: 대시보드 상단(**KPI 카드 + 차트 3개**)을 **PNG 이미지**로 저장합니다.
- **PDF 내보내기**: 같은 영역을 **가로 A4 PDF**로 저장합니다. (내용이 길면 여러 페이지로 자동 분할)
- 두 기능 모두 **하단의 전체 목록 표는 제외**하고 KPI·차트까지만 저장됩니다.
- 파일명에는 저장 날짜가 자동으로 붙습니다. (예: `가스연료기술센터_대시보드_20260616.png`)

> 라이브러리: [html2canvas](https://html2canvas.hertzen.com/) · [jsPDF](https://github.com/parallax/jsPDF) (CDN)

## 📁 엑셀 파일 요구사항

다음 **3개 시트**가 있어야 합니다. (시트 이름이 정확히 일치해야 합니다.)

- `국가연구개발사업`
- `수탁용역`
- `시험인증`

추가로 목표·달성률을 보려면 다음 시트가 필요합니다. (선택)

- `분기별 보고` — "운용자금 목표금액" 항목 포함

### 💰 금액(실적) 기준

| 시트 | 실적으로 사용하는 값 |
|------|------------------|
| 국가연구개발사업 | **당해년도 사업비 합계** (인건비 + 간접비) |
| 수탁용역 | **총 인정실적(A+B)** |
| 시험인증 | **시험수수료(A)** — 전부 **극저온 팀**으로 집계 |

### 규칙

- **헤더 자동 인식**: 제목 행이나 2단(병합) 헤더가 있어도 키워드로 헤더 행을 자동 탐지합니다.
- **합계 행 자동 제외**: 시트 맨 아래 "합계/총계/소계" 행은 제외하고 직접 재합산합니다.
- **빈 값 = 0원 처리**: `미정`, `협약전`, `미승금`, `-`, 빈칸 등 **숫자로 변환할 수 없는 값은 모두 0원**으로 계산됩니다. (나중에 값이 채워지면 자동 반영)

## 🎨 표 표시 규칙

- **긴 텍스트 열**(과제명 · 시험/용역 계약명 · 시험/용역항목)은 넓은 너비 + **왼쪽 정렬**, 한 줄로 표시하며 넘치면 말줄임(…) 처리 후 마우스를 올리면 전체 내용을 보여줍니다.
- **금액 열**(사업비 합계 · 인정실적 · 시험수수료)은 **오른쪽 정렬**.
- 그 외 열과 **헤더(첫 줄)**는 **가운데 정렬**.
- 표 맨 위에 **합계 행**(`총 00건` + 총계 금액)이 헤더 바로 아래에 고정(sticky)됩니다.
